HTML Cheat Sheet

CSS Selectors and Specificity

Selector Description Comparative Priority
h1 Selects all h1 html elements This would overwrite * (or all) and body selectors
h1.special Selects all h1 elements with the .special class This would overwrite the above h1 element
#example This would select the element with the id="example" This would overwrite all prior selectors. It is intended to only use on one HTML element
Inline Styles Applied specifically to an element in HTML Inline styles always overwrite any styles in external style sheets
!important The !important rule is used on a singular style declaration This overwrites all other declarations, but is generally considered bad practice

Further Remarks

It is best practice to use semantic CSS and have descriptive classes. Use IDs when a uniqe style is necessary for a singular element. Avoid using the !important rule whenever possible, as it can make debugging difficult. You should also strive to write efficient CSS and avoid unnecessary duplication, using a combination of classes to achieve the styling desired.
